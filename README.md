# CPU-Scheduling

## Installations

1. Clone repository:
```
git clone https://github.com/Kammar1006/CPU-Scheduling.git
cd CPU-Scheduling
```

2. Set up the virtual environment:
```
python -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\activate`
```

3. Install dependencies:
```
pip install numpy
```

4. Run file:
```
python FIFO_SJF.py
python LRU_LFU.py
```
